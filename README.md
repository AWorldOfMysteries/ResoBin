# [`ResoBin`](https://resobin.netlify.app/) - Easily Share Course Material

## Requirements

* React 16.8+

## Getting started

* Install required modules:

    ```powershell
    yarn install
    ```

* Run development server:

    ```powershell
    yarn start
    ```

* Frontend server runs at [`http://localhost:3000`](http://localhost:3000)
